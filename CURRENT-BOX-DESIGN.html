Follow the Box 当前盒子设计方案

1. 产品形态

首台样机采用“控制盒 + 外部电机控制器”结构：

控制盒：ESP32-S3 / UWB / DS600 接收机 / TOF / IMU / DC-DC / 急停链 / 接口端子
外部驱动：左/右 36V 350W 无刷控制器
视频：独立 ESP32-S3-CAM 模块

原因：控制核心和大电流驱动分开，布线更安全，后续换底盘时只改驱动适配和 Profile。

2. 外观布局

顶部

位置放什么目的
顶部最高处UWB 外置天线保持无遮挡，远离金属和动力线
顶部容易拍到的位置蘑菇头急停任何人一眼看到、一手拍下
顶部或前上方状态灯/导光柱显示上电、手动、跟随、故障、急停
顶部可选提手/安装耳方便搬运和维护

UWB 天线周围 50mm 内尽量不要有金属、电池、大电流线、DC-DC Buck 电感/模块；若盒体较小，UWB 天线优先外置到顶部。

前面板

位置放什么说明
前中TOF 中路窗口/端口正前方障碍
左前/右前TOF 左/右窗口/端口判断绕行空间
前上方摄像头窗口/支架H5 远程看车前画面
前面板FRONT 方向箭头防止盒子装反

后面板

位置放什么说明
后下方BAT IN电池输入，线最短
后下方CTRL L / CTRL R左右无刷控制器信号线
后下方MOTOR/POWER 走线出口动力线远离 UWB 和传感器
后侧保险丝维护口方便更换，避免手碰裸铜

左右侧

位置放什么说明
左侧US LEFT左侧超声端口
右侧US RIGHT右侧超声端口
侧面USB 调试/维护口开发阶段保留，产品版加防水盖
侧面蜂鸣器/扬声器孔声音外放，避开麦克风

3. 盒内分区

┌────────────────────────────────────────────┐
│ 低压控制区：ESP32 / DS600 RX / UWB / IMU      │  ← 上层，远离电机线
├────────────────────────────────────────────┤
│ 传感器接口区：TOF / 超声 / I2C / UART 端口     │  ← 前侧/左右侧
├────────────────────────────────────────────┤
│ 电源区：DC-DC / F2 / F3 / 分压 / 端子排        │  ← 靠近 BAT IN，远离 UWB
├────────────────────────────────────────────┤
│ 信号驱动区：PWM→0-5V ×2 / 光耦/MOS / CTRL端子 │  ← 靠近控制器线出口
└────────────────────────────────────────────┘

当前首台样机不建议把两个 350W 控制器塞进小控制盒；控制器可贴底盘或独立固定，便于散热和走粗线。

4. 尺寸建议

版本建议尺寸适合内容
信号控制盒160-220mm × 120-160mm × 60-90mmESP32、UWB、DC-DC、接口板、保险、PWM转模拟模块
动力一体盒240-350mm × 160-240mm × 90-140mm控制器、接触器、大电流端子、散热风道

首台建议先做信号控制盒；大功率控制器放盒外或底盘散热位置。

5. 材料选择

材料建议
ABS/PC 塑料盒首台更友好，射频不容易被屏蔽，开孔方便
铝合金盒强度和散热好，但 UWB/WiFi 需要外置天线或塑料窗口
铝盒 + 塑料天线窗产品版更好，结构复杂一点
3D 打印外壳适合临时验证，不适合作为长期户外外壳

6. 端口标签

建议盒子外部直接印或贴这些标签：

BAT IN
ESTOP
CTRL L
CTRL R
UWB ANT
TOF L / TOF C / TOF R
US LEFT / US RIGHT
CAM
USB DEBUG
5V OUT
GND

每个端口都要有方向、防呆、线号；不能只靠颜色记忆。

7. 装配规则

  1. 低压板固定在绝缘柱上，不靠杜邦线吊着。
  2. DC-DC 预留散热空间，先空载调到 5.0V 再接负载。
  3. 动力线和相线不要与 UWB、TOF、PWM 油门线捆在一起。
  4. DC-DC Buck 与 UWB 模块/天线保持至少 50mm，5V 输出端并联低 ESR 电解电容 + 0.1uF 陶瓷电容。
  5. 主负极使用星型汇流：两个控制器 BAT-、电池 BAT-、DC-DC VIN- 走粗线到主负极点；ESP32/传感器/转把 GND 只作低压信号参考，不能成为控制器 BAT- 断线后的回流路径。
  6. 外壳醒目位置贴“上电后静止 3 秒”标识，避免 JY60/JY61P 上电零偏标定期间推晃导致 yaw 漂移。
  7. 穿盒孔必须加护线圈，防止割线。
  8. 外露传感器线必须有应力释放。
  9. 每根线两端贴线号，拍照存档。
  10. 盒内预留手能插拔保险和 USB 的空间。
