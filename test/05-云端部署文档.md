## 四、Nginx 配置（已配置，仅用于参考）

文件路径：`/www/server/panel/vhost/nginx/extension/boonai.cn/followbox-cloud.conf`

```nginx
location /fb/ {
    proxy_pass http://127.0.0.1:8080/;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;

    # SSE 支持（遥测实时流）
    proxy_buffering off;
    proxy_cache off;
    proxy_read_timeout 86400s;
    proxy_send_timeout 86400s;
}
```

## 五、API 路由

| 方法 | 路径 | 说明 |
|---|---|---|
| `GET` | `/fb/` | H5 前端页面 |
| `POST` | `/api/device/{id}/ingest` | ESP32 上传遥测数据（需 `token` 参数） |
| `GET` | `/api/device/{id}/events` | SSE 实时遥测流 |
| `GET` | `/api/device/{id}/command` | ESP32 拉取待执行指令 |
| `POST` | `/api/device/{id}/command` | 操作员发送遥控指令（需 `Authorization: Bearer`） |

## 八、注意事项

### 8.1 WSL 路径转换

WSL 中使用 `/mnt/d/` 代表 Windows 的 `D:\`：

```bash
D:\car\Follow the box\cloud\  →  /mnt/d/car/Follow the box/cloud/
```

### 8.2 sandbox 文件截断问题

Hermes sandbox 存在文件内容截断限制。写入长 token 时，使用 Python 或 `hermes_tools write_file` 替代 bash `echo/scp`，确保文件内容完整。

### 8.3 摇杆参数

当前摇杆在 `app.js` 中实现，尺寸参数：

| 断点 | 摇杆容器 | 摇杆手柄 |
|---|---|---|
| 桌面 (>640px) | 180×180px | 50×50px |
| 平板 (420-640px) | 150×150px | 44×44px |
| 手机 (<420px) | 130×130px | 38×38px |

### 8.4 依赖

`server.js` 纯 Node.js 标准库，无需外部 npm 包。

## 九、快速部署脚本

一键部署脚本，保存为 `deploy-followbox.sh`：

```bash
#!/bin/bash
# FollowBox 云端一键部署脚本

LOCAL_BASE="/mnt/d/car/Follow the box/cloud"
REMOTE="root@82.156.85.60:/www/wwwroot/followbox-cloud"
SSH_OPTS="-o IdentitiesOnly=yes -o PubkeyAcceptedAlgorithms=+ssh-rsa -o HostkeyAlgorithms=+ssh-rsa -i /mnt/c/Users/陈雨/Downloads/codex.pem -P 51400"

echo "=== 上传后端文件 ==="
scp $SSH_OPTS "$LOCAL_BASE/server.js" "$LOCAL_BASE/package.json" "$REMOTE/"

echo "=== 上传前端文件 ==="
scp $SSH_OPTS "$LOCAL_BASE/public/index.html" "$LOCAL_BASE/public/app.js" "$LOCAL_BASE/public/style.css" "$REMOTE/public/"

echo "=== 安装依赖并重启 ==="
ssh $SSH_OPTS root@82.156.85.60 "
  cd /www/wwwroot/followbox-cloud && \
  npm install && \
  pm2 restart followbox-cloud
"

echo "=== 验证 ==="
curl -s -o /dev/null -w 'HTTP %{http_code}\n' 'https://www.boonai.cn/fb/'
echo "部署完成"
```
