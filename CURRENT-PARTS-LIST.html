Follow the Box 当前配件清单

1. 当前样机架构

HOTRC DS600 接收机 PWM
  -> ESP32-S3-DevKitC-1
  -> PWM 转 0-5V 模拟量模块 ×2
  -> 左/右 36V 350W 有霍尔无刷控制器
  -> 左/右轮毂电机

辅助模块：

UWB GC-P2304-GS-2 -> 跟随距离/方位
VL53L1X ×3 + TCA9548A -> 前左/前中/前右避障
HC-SR04 ×2 -> 左/右侧辅助避障
JY61P -> 姿态/yaw 阻尼
ESP32-S3-CAM + OV5640 -> 独立视频，不参与安全控制

数量原则：首台样机按实际安装位购买，不默认多买备品。
