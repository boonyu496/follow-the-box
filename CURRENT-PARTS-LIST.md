# Follow the Box 当前配件清单

## 1. 当前样机架构

```text
HOTRC DS600 接收机 PWM
  -> ESP32-S3-DevKitC-1
  -> PWM 转 0-5V 模拟量模块 ×2
  -> 左/右 36V 350W 有霍尔无刷控制器
  -> 左/右轮毂电机
```

辅助模块：

```text
UWB GC-P2304-GS-2 -> 跟随距离/方位
VL53L1X ×3 + TCA9548A -> 前左/前中/前右避障
HC-SR04 ×2 -> 左/右侧辅助避障
JY61P -> 姿态/yaw 阻尼
ESP32-S3-CAM + OV5640 -> 独立视频，不参与安全控制
```

数量原则：首台样机按实际安装位购买，不默认多买备品。

## 5. 到货后必须确认

| 项目 | 需要确认 |
|---|---|
| DS600 接收机 | S/+/- 排针顺序、供电范围、PWM 高电平、失控保护 |
| 无刷控制器 | 电门锁、转把、刹车、倒车、学习、三速、巡航等线色和插头 |
| 轮毂电机 | 三相线、霍尔线序、左右方向 |
| 电池 | 铭牌、容量、BMS 持续/峰值放电能力；若换 48V/54.6V 底盘，确认 ADC 已为 220k/10k 分压 |
| 急停 | 背面端子编号、必须确认第二触点或隔离检测方案给 GPIO21 |
